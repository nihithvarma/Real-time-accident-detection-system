# Real-time-accident-detection-system
Real-tiem=me accident detection system using YOLOv11 for identifying road accidents from video streams with high accuracy and fast inference.


## Features

* Real-time accident detection from video streams
* High-speed object detection using YOLOv11
* Bounding box visualization for detected accidents
* Classification of accident scenarios
* Supports both live camera feed and video input

---

## Technologies Used

* Python
* YOLOv11
* OpenCV
* NumPy
* Deep Learning
* Pytorch
* Matplotlib
* Seaborn
---

## How It Works

1. Input video is captured from a live camera or video file
2. Frames are processed in real-time
3. YOLOv11 model detects objects and identifies accident patterns
4. Detected accidents are highlighted with bounding boxes
5. Output is displayed with detection results

---

## Installation and Usage

### 1. Clone the repository

```
git clone https://github.com/nihithvarma/Real-time-accident-detection-system.git
cd accident-detection-yolov11
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the project

```
python main.py
```
